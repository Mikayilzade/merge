# Merge Arcana

Мобильный offline-first прототип самостоятельной merge-tactics игры. Игрок покупает бойцов, расставляет их на поле, объединяет одинаковые карты и наблюдает автоматический бой против ИИ.

## Что уже работает

- забег из 10 раундов с тремя жизнями;
- магазин, золото, проценты за накопления и бонус серии;
- поле 3×3, скамья на 5 мест и растущий лимит отряда;
- объединение одинаковых бойцов от 1★ до 3★;
- шесть оригинальных бойцов, три синергии и простой боевой ИИ;
- полноценная анимация автоматического боя;
- конечная прокачка карт до 10 уровня;
- бесконечный уровень заклинателя с +0,1% к характеристикам за уровень;
- артефакт на +10%, который можно активировать перед выбранным раундом;
- интерактивное сравнение бесплатной версии и подписки;
- локальное сохранение прогресса;
- PWA: после публикации можно установить на домашний экран iPhone;
- работа без интернета после первого открытия.

## Локальный запуск

Проект не требует сборки и внешних библиотек.

```bash
python3 -m http.server 8080
```

Затем открыть `http://localhost:8080`.

## Проверки

```bash
npm test
npm run check
```

Тесты запускаются локально. GitHub Actions в репозитории намеренно не добавлены, чтобы не создавать лишние уведомления во время первой итерации.

## Публикация через GitHub Pages

После слияния PR: **Settings → Pages → Deploy from a branch → main / root**. После первой публикации игра будет доступна по адресу вида `https://mikayilzade.github.io/merge/`.
