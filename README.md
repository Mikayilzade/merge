# Merge Arcana

Мобильная offline-first merge-tactics игра. Игрок покупает героев во временном магазине забега, расставляет их на поле 3×3, объединяет одинаковых бойцов и наблюдает автоматические 2,5D бои.

## Версия 0.2

Главная цель обновления — сделать игру понятной без догадок:

- 8 персонажей с индивидуальной векторной графикой;
- точные характеристики: здоровье, удар/лечение, DPS, частота атак, движение, дальность, броня и крит;
- способности, цитаты и лор каждого героя;
- 4 синергии с точными числовыми эффектами;
- 2,5D арена, анимации атак, снаряды, взрывы, лечение, критические удары и горение;
- отдельная внешняя лавка постоянного прогресса;
- более информативный внутренний магазин забега;
- три косметические темы арены;
- сохранение прогресса версии 0.1 без сброса.

## Запуск

Игра не требует сборщика или внешних библиотек. Достаточно открыть `index.html` через локальный HTTP-сервер.

```bash
python3 -m http.server 4173
```

Затем открыть `http://localhost:4173`.

## Локальная проверка

```bash
npm run check
npm test
```

GitHub Actions намеренно не добавлены. Проверки запускаются локально, чтобы не создавать лишние уведомления.

## Публикация

GitHub Pages публикует содержимое ветки `main` из корня репозитория. После слияния обновления сайт доступен по адресу:

`https://mikayilzade.github.io/merge/`

При старом закэшированном интерфейсе достаточно полностью закрыть установленное веб-приложение и открыть снова. Service Worker версии 0.2 удаляет старый кэш.
