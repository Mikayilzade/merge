# Roadmap

## 0.2 — Readable combat and identity

- [x] точные характеристики и формулы;
- [x] 8 героев и 4 синергии;
- [x] способности и лор;
- [x] векторная 2,5D графика;
- [x] внешний и внутренний магазины;
- [x] косметические арены;
- [x] локальные тесты без GitHub Actions.

## 0.3 — Итерация по игровому фидбеку

- баланс отдельных героев;
- скорость и читаемость боя;
- новые способы объединения;
- более умный ИИ и контр-составы;
- журнал боя с реальным уроном/лечением каждого героя;
- задания и первые достижения.

## 0.4 — Долгосрочный прогресс

- несколько Архонтов с разными стилями;
- артефакты с активными эффектами вместо единого +10%;
- сезонные правила без pay-to-win преимущества;
- полноценная система косметики.

## 0.5 — Социальный прототип

- асинхронные «призрачные» команды друзей;
- коды приглашений;
- повтор боя;
- подготовка серверной модели.

## 1.0 — Онлайн

- живой PvP;
- серверная авторитетная симуляция;
- матчмейкинг;
- защита от читов;
- публикация в магазинах приложений после проверки удержания.
